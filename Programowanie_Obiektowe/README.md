Przedmiot z drugiego semestru w ramach którego wykonano dwa projekty:

-PO_C
  Projekt napisany w języku c++ (obiektowo) obsługiwany w terminalu.
  W ramach projektu wykonano grę turową w której każdy organizm wykonuje daną akcję zdefiniowaną przez instrukcję. Gracz dodatkowo ma umiejętność "Magiczny Eliksir" która zwiększa jego siłę do 10 przez określony czas. Program oprócz zwykłej rozgrywki pozwala na zapisywanie/wczytywanie i podgląd logów.
  Szczegóły dotyczące zadania są w pliku "projekt_PO_wirtualny_swiat_2017".
  Program uruchamiany w programie Visual Studio.
  Sterowanie:
  PORUSZANIE 'w','s','a','d'
  UMIEJETNOSC 'u'
  WYJDZ 'q'
  ZAPISZ 'z'
  WCZYTAJ 'x'
  NOWA_GRA 'n'
  LOGI 'l'
  
-PO_JAVA
  Projekt napisany w języku java z interfejsem graficznym z biblioteki Swing.
  W ramach projektu napisano grę turową w języku java i dodano interfejs graficzny. Dodatkowo można dodawać organizmy klikając puste pola oraz dodano wariant planszy hex.
  Szczegóły dotyczące zadania są w pliku "projekt_PO_wirtualny_swiat_2017".
  Program uruchamiany w programie IntelliJ
  Sterowanie:
  PORUSZANIE 'w','s','a','d','q','e'
  PLANSZA KWADRATOWA 'K'
  PLANSZA HEX 'H'
  WYJDZ 'q'
  WCZYTAJ 'x'
  NOWA_GRA 'n'
